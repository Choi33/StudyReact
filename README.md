## 리액트로 만드는 TODO📄

1. 시작이 반이다.

   - 2021년 3월 5일

     🌱 갑자기 리액트를 공부해야겠단 생각이 들기 시작했다. 사실 몇 달 전 부터 생각하고 있던 일이었다. 하지만 늘 마음만 있을 뿐, 실행력은 없었다. 이러다간 평생 리액트 공부해야지 라는 말만 하고 다닐 것 같아서 친한 선배에게 '주변에 리액트 책 추천해줄 사람 있어?' 라고 물어봤더니 선배는 웃으면서 리액트를 잘하는 자신의 지인을 나에게 소개해주었다. 책을 보며 가볍게 시작하려 했던 리액트 공부는 격변의 소용돌이 속으로 떨어지고 있었다.

     

   - 2021년 3월 8일

     🌱 친한 선배가 소개해준 지인분과 가볍게 이야기를 나누었다. 지인분은 내 상황을 듣고는 Todo 리스트를 한 번 가볍게 만들어보는게 어떻겠냐고 말했다. 리액트의 존재만 알고 문법은 전혀 모르는데 만들 수 있을까? 라는 생각을 문뜩 들었다. 하지만 대학 생활 동안 여러가지 새로운 언어를 접하고 배워가면서 한 가지 깨달은게 있다. 바로 '새로운 언어를 배울 땐 간단한 프로젝트를 만들면서 익혀가는게 좋다' 이다. 물론 해당 방법은 기본적인 문법적 지식이 있단 가정하에 먹힌다. 가령 생각해봐라, C언어가 프로그래밍 언어라는 사실만 아는 사람에게 C언어를 이용해 좌석배치 프로그램을 만들어내라 라고 하면 #include도 하지 못할 것이다.  하지만 개인적으로 Java Script, CSS, HTML, Webpack 등으로 웹페이지를 만든 경험이 있었다. 그러니 어떻게든 하다보면 되지 않을까? 라는 생각으로 git 저장소를 파고 구글과 node.js의 도움을 받아 리액트를 설치했다.
